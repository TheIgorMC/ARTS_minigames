Place your scene folders here (e.g. [01]Scene Name/, [02]Another Scene/).
Each folder can contain:
  - [BG]filename.png   = Background image
  - [MOOD]filename.png  = Mood overlay image
  - item images         = Draggable table objects

A [00]Global folder is created automatically for shared assets.
A [CHAR] Character pictures folder holds character art.
